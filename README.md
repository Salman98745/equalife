# equalife
EquaLife — IMU sensorlarından alınan triqonometrik verilənlərin ($Roll/\phi, Pitch/\theta$) analizi ilə təzyiq yaralarını önləyən AI sistemidir. "Sabahın Alimləri" müsabiqəsi (Riyaziyyat kateqoriyası) üçün hazırlanıb. Euler bucaqları və Qərar Ağacı (Decision Tree) alqoritmi ilə %98.4 dəqiqliklə real-zamanlı monitorinq təmin edir.
